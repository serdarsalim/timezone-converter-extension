# Times

Minimal Chrome extension for inline timezone comparison.

## What it does

- Inline edit city names and timezones from the main popup
- Inline set a reference time from any city and recalculate every other row from it
- Reset the active reference time without opening settings
- Drag cards to reorder them
- Add cities from the main view
- Toggle between `12h` and `24h` from the header settings control
- Show a sun icon during daylight hours and a moon icon at night

## Load in Chrome

1. Open `chrome://extensions`
2. Turn on Developer mode
3. Click `Load unpacked`
4. Select `/Users/slm/my-portfolio/timezone-extension`
